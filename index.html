Die Programmiersprache Java Script

JavaScript ist eine der am häufigsten verwendeten Programmiersprachen und wird von Millionen von Entwicklern auf der ganzen Welt eingesetzt. Es ist eine leicht zu erlernende Sprache und bietet viele leistungsfähige Funktionen, einschließlich der Möglichkeit, Funktionen als Objekte zu behandeln, um Vererbung und Polymorphie zu ermöglichen, sowie die Möglichkeit, asynchrone Programmierung zu ermöglichen, die die Leistungsfähigkeit von Webanwendungen verbessert.

Eine kurze Geschichte einer Sprache

Die Entwicklung von JavaScript geht zurück auf das Jahr 1995, als Brendan Eich bei Netscape Communications Corporation arbeitete. Netscape hatte zu dieser Zeit den Browsermarkt beherrscht und plante, einen neuen Browser namens "Mozilla" zu entwickeln. Eich wurde beauftragt, eine Skriptsprache für den neuen Browser zu erstellen, die es ermöglichen würde, Webseiten interaktiver zu gestalten. Die erste Version von JavaScript wurde innerhalb von nur zehn Tagen von Eich entwickelt und im Dezember 1995 zusammen mit dem ersten öffentlich verfügbaren Browser von Netscape veröffentlicht. Anfangs wurde die Sprache "LiveScript" genannt, aber schon bald wurde der Name in "JavaScript" geändert, um von der Popularität von Sun Microsystems' Java zu profitieren. Im Jahr 1996 reichte Netscape den Quellcode von JavaScript bei der European Computer Manufacturers Association (ECMA) ein, um die Sprache als Standard zu formalisieren. Dies führte zur Entwicklung des ECMAScript-Standards, der erstmals im Jahr 1997 veröffentlicht wurde. Seitdem hat der Standard mehrere Revisionen durchgemacht, wobei ECMAScript 6 im Jahr 2015 die bislang umfassendste und umfangreichste Version war. In den späten 1990er Jahren wurde JavaScript von anderen Browserherstellern übernommen, darunter Microsoft mit dem Internet Explorer. Mit der zunehmenden Verbreitung von JavaScript wuchs auch die Nachfrage nach besseren Tools und Frameworks. Die Open-Source-Community entwickelte daraufhin zahlreiche JavaScript-Bibliotheken und -Frameworks, darunter jQuery, React und Angular. In den letzten Jahren hat sich JavaScript zunehmend auch auf der Server-Seite durchgesetzt, insbesondere durch die Verwendung von Node.js, das es ermöglicht, JavaScript-Code auf dem Server auszuführen und so die Erstellung von schnellen und skalierbaren Webanwendungen zu ermöglichen.

Die 10 wichtigsten Begriffe

  1. Variablen Speichern von Werten wie Strings, Zahlen oder Booleans.
  2. Datentypen Verschiedene Arten von Daten, wie Strings, Zahlen, Booleans, Arrays, Objekte, Null und Undefined.
  3. Funktionen Wiederverwendbare Codeblöcke, die eine bestimmte Aufgabe ausführen.
  4. Bedingungen Eine Möglichkeit, auf bestimmte Bedingungen zu reagieren und entsprechenden Code auszuführen.
  5. Schleifen for-, while-, und do-while-Schleifen, die es ermöglichen, Codeblöcke mehrmals auszuführen.
  6. Arrays Eine Möglichkeit, mehrere Werte in einer einzigen Variablen zu speichern.
  7. Objekte Mehrere Eigenschaften und Methoden werden in einer einzigen Variablen gespeichert.
  8. DOM Das Document Object Model greift auf den Inhalt, die Struktur und das Styling von HTML-Dokumenten zu.
  9. Events Aktionen, die vom Benutzer ausgelöst werden, wie das Klicken auf einen Button oder das Scrollen einer Seite.
  10. Asynchrone Programmierung Führt Code aus, während auf langsame Vorgänge wie das Laden von Daten gewartet wird.
